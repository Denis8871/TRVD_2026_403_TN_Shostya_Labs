**Технологічний стек**

- **Front-end:** React 18 (Vite), React Router, Lucide Icons. 
- **BaaS (Backend):** Supabase (PostgreSQL, Auth, RLS). 
- **API:** PostgREST (автоматично згенероване Supabase). 

**Структура репозиторію**

- frontend/ — клієнтська частина додатку (React + Supabase SDK). 
- lab1/ — аналіз вимог, User Stories та NFR. 
- lab2/ — архітектура (C4 модель) та ER-діаграма бази даних. 
- lab3/ — реалізація DAL, Service Layer та опис API ендпоінтів. 
- lab4/ — налаштування безпеки: автентифікація, ролі та RLS політики. 
- lab5/ — опис інтерфейсу, UX/UI фіч та інтеграції. 

**Основний функціонал**

- **Керування задачами:** Створення, редагування, зміна статусу (todo/done) та видалення. 
- **Пошук та фільтрація:** Миттєвий пошук за назвою та фільтри за пріоритетом. 
- **Безпека:** Захищені маршрути (Protected Routes) та ізоляція даних на рівні БД (RLS). 
- **Адмін-панель:** Перегляд списку користувачів та можливість їх блокування (is\_blocked). 
- **UX:** Підтримка темної теми (Dark Mode) та адаптивний дизайн. 

**Швидкий запуск**

**1) Налаштування середовища**

Створіть файл .env.local у папці frontend/ та додайте ваші ключі Supabase:

Plaintext

VITE\_SUPABASE\_URL=ваш\_url

VITE\_SUPABASE\_ANON\_KEY=ваш\_ключ

**2) Запуск клієнта**

Bash

cd frontend

npm install

npm run dev
